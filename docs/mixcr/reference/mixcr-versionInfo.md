# `mixcr versionInfo`

Output information about MiXCR version which generated the file.

```
mixcr versionInfo 
    [--no-warnings] 
    [--verbose] 
    [--help] 
    input_file.(vdjca|clns|clna|shmt)
```

`-nw, --no-warnings`
: Suppress all warning messages.

`--verbose`
: Verbose warning messages.

`-h, --help`
: Show this help message and exit.
