# `mitool process-fastq`

Usage: 
```shell
mitool process-fastq [OPTIONS] SOURCE DESTINATION
```
Arguments:
`SOURCE`
: Input *.fastq[.gz] file

`DESTINATION`
: Output *.fastq[.gz] file

Options:

`-e, --expression <TEXT>`
: Pipeline expression.

`-n, --limit <INT>`
: Limit number of reads

`-h, --help`
: Show this message and exit
